\appendix
\section{Фрагменты кода программы <<Вычисление факториала>>}
\label{app:factorial}
Функция вычисления факториала:

\begin{minted}[style=bw,
 linenos=true,
 breaklines=true,
 numbersep=5pt,
 tabsize=2,
 fontsize=\small,
 bgcolor=white]{cpp}
long long fact(long long N) {
	if (N < 0) //отрицательное число
		return -1;
	else if (N == 0 || N == 1) // 0! = 1
		return 1;
	else
		return N * fact(N - 1); //n! = n * (n - 1)!
}
\end{minted}
